# 添加RSSI数据过滤功能计划

## 1. 在主窗口中添加RSSI过滤按钮

* 在协议选择区域添加一个RSSI过滤按钮

* 按钮文本应显示当前状态（"开启RSSI过滤"或"关闭RSSI过滤"）

* 按钮点击时切换过滤状态

## 2. 添加过滤逻辑

* 在MainWindow类中添加一个rssi\_filter\_enabled变量，用于存储当前过滤状态

* 在on\_data\_received方法中添加过滤逻辑，当过滤功能开启时，忽略RSSI命令的数据

* 确保其他命令不受影响

## 3. 更新UI状态

* 当过滤状态改变时，更新按钮文本以反映当前状态

* 在日志中记录过滤状态的变化

## 4. 重新打包应用程序

* 运行setup.py脚本重新打包

* 验证生成的exe文件能正常运行

## 5. 验证所有修改

* 确保过滤按钮能正确开启和关闭RSSI过滤功能

* 确保其他命令的数据接收不受影响

## 实现步骤

1. 修改ui/main\_window\.py文件，添加RSSI过滤按钮
2. 在MainWindow类中添加rssi\_filter\_enabled变量
3. 在on\_data\_received方法中添加过滤逻辑
4. 运行setup.py脚本重新打包应用程序
5. 测试生成的exe文件
